🎯 Why Attendance Tracker?

Managing attendance manually is time-consuming, repetitive, and prone to calculation errors.

Attendance Tracker was created to provide a modern, reliable, and easy-to-use solution for attendance management without requiring complex software or internet connectivity.

The project focuses on:

- Simplicity
- Reliability
- Accessibility
- Security

while remaining lightweight enough to run on almost any device.

🧠 Design Philosophy

Attendance Tracker follows three simple principles:

1. Simplicity

The interface should be easy enough for any student or teacher to use without training.

2. Reliability

Attendance calculations must remain accurate and consistent.

3. Accessibility

The system should work efficiently across:

- Mobile Phones
- Tablets
- Desktop Computers

🔐 Security Notice

Attendance Tracker includes:

- Password Protected Access
- Session Tracking
- Secure Logout
- Password Change Functionality

All attendance data remains stored locally on the user's device.

No external servers or cloud databases are used.
